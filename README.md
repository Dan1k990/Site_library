# BookReader

Информационная система для хранения, каталогизации и онлайн-чтения электронных книг.

---

## Описание проекта

**BookReader** — сайт для:
- просмотра и чтения электронных книг;
- поиска книг по каталогу;
- сортировки и фильтрации по жанрам и авторам;
- учета посещаемости;
- уведомлений и аналитики загрузки.

Система ориентирована на:
- читателей;
- администраторов библиотеки;
- руководство сайта.
